# 🐾 PanelPug

## The community directs it, AI draws it

Right now most AI projects are super singleplayer, and super boring. This project is an experiment in something more fun - a shared canvas where a bunch of people can collaborate to tell stories together.

**PanelPug** is an attempt to build a structure forum for community-steered generative web comics. A place for folks to theorize, and discuss, and ultimately quest.

And to do that well, it shouldn't just be the platform that's open, but the code too. It needs to be steerable in the same way so that the community can steer how the platform itself evolves, in addition to individual stories.

## What we're building

PanelPug is a platform for folks to generate "forum quest" style web comics, where humans provide the direction, and AI makes it concrete. Players can suggest actions for the characters, which are then voted on by the community, and then the top option becomes the next segment of the story.

## Community guidelines

1. PanelPug is a global studio. We have zero tolerance for hate speech, discrimination, or gatekeeping based on identity or experience level. Whether you’re a senior dev or a first-time "marker holder," you have a spot at the table.
2. We are experimenting with AI, which is a lot of new territory. This means we don't use the platform to generate non-consensual imagery, deepfakes, or content designed to cause real-world harm.
3. Document. Document. Document. Having conversations and discussing in the open helps us learn and evolve over time.

## Getting started

Grab a seat at the director's table. We organize on [Discord]() to figure out what to build next. Popular discord threads get triaged by humans and turned into code updates (PRs) by AI, which is then human reviewed*. Hop over there to contribute your thoughts and perspective.

Or if you want to start prototyping a new idea, clone the repo, run the /onboard AI workflow and toss up PR for others to play around with!

*To make sure we're operating safely, all PRs are currently reviewed by @BeksOmega, but in the future I'd love to add more folks to the crew!
